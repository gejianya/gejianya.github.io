Hexo + 阿里云 & GitHub 搭建个人博客本地部署 GitHub 部署阿里云部署准备工作安装必要的软件 Node.jshttps://nodejs.org/en/ Windows：通过 nvs（推荐）或者 nvm 安装。

准备工作

安装必要的软件

Node.js

https://nodejs.org/en/ （直接点击本链接就有Mac版）

  • Windows：通过 nvs（推荐）或者 nvm 安装。
  • Mac：使用 Homebrew 或 MacPorts 安装。
  • Linux（DEB/RPM-based）：从 NodeSource 安装。
  • 其它：使用相应的软件包管理器进行安装，可以参考由 Node.js 提供的 指导。

git

https://git-scm.com/

  • Windows：下载并安装 git.
  • Mac：使用 Homebrew, MacPorts 或者**下载 安装程序**。
  • Linux (Ubuntu, Debian)：sudo apt-get install git-core
  • Linux (Fedora, Red Hat, CentOS)：sudo yum install git-core

Mac系统获取本机公钥

  1. 首先用 ssh -vT git@github.com 测试一下，会去你用户目录下面找公钥文件, 有打印出下面的信息,这说明/User/用户名/.ssh/ 下面去寻找公钥文件，当前用户目录下面的确没有公钥文件。

  2. 执行: cd /Users/用户名/.ssh/ 如：cd /Users/gejianya/.ssh/

  3. 执行: ssh-keygen -t rsa -b 4096 -C “Git 上关联的email” 如：ssh-keygen -t rsa -b 4096 -C “gjywin@163.com”

点击回车

输入y

生成id_rsa.pub 文件后, 执行以下命令：vi id_rsa.pub

自动打开文件, 复制文件内容（SSH keys）到GitHub–> settings–>SSH and GPG Keys –> new SSH key，然后保存。

  1. 重新执行 ssh -vT git@github.com , 成功提示:

将 SSH keys 保存到 Github

打开 github 点击头像 ->settings, 在设置页面找到 SSH and GPG keys，单击 New SSH key 新建 SSH KEY。
